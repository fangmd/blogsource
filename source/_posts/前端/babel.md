# 基本知识

## 配置文件格式

支持多种格式的配置文件

1. `babel.config.js`

优点：可以使用 js 动态生成配置

2. `.babelrc`

## 配置内容

1. `presets`

设置转码规则

2. `plugins`

## babel-cli 使用

全局安装 `babel-cli`

```
npm install -g @babel/core @babel/cli
```

>babel 所需要的库都在 @babel 下(babel 7 之后)
