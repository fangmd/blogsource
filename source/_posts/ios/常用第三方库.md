# 键盘管理

IQKeyboardManager

![https://github.com/hackiftekhar/IQKeyboardManager](https://github.com/hackiftekhar/IQKeyboardManager)

作用：

1. 键盘弹出的时候管理布局的变化（不遮盖布局）
2. 多个输入框快速切换

## 基本使用

在 AppDelegate 中开启：

```
import IQKeyboardManagerSwift

IQKeyboardManager.sharedManager().enable = true

```
