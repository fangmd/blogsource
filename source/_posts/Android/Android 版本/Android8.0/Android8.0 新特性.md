新特性：

1. Picture-in-picture
2. Notification dots
3. Notification channels
4. Autofill framework
5. Autosizing TextView
6. Downloadable fonts
7. Adaptive icons
8. Shortcut pinning
9. Wide-gamut color
10. WebView features
11. Java 8 language APIs
12. Media features
13. Multidisplay support
14. Neural Networks API
15. Android Oreo (Go edition)
